# Онлайн-библиотека (тестовое задание)
Посмотреть результат можно здесь: https://testwork-library.herokuapp.com/

Последовательность шагов для локального запуска:</br> 
<b>npm install</b> - установка зависимостей</br> 
<b>npm start</b> - запуск приложения (порт 3000)

## Требования по функционалу

Создать json файл с массивом сущностей книг, содержащих идентификатор, название, описание и другие дополнительные данные. </br> 
Реализовать сервер на node.js , возвращающий данные посредством запросов: </br> 
/api/books - получение всего массива книг </br> 
/api/book/id - получение определённой книги из массива </br> 
Реализовать клиентское Single-Page приложение с просмотром списка книг и определённой книги по идентификатору. 

## Требования по технической части

Ипользуемые технологии:</br> 
Webpack, React, Redux
